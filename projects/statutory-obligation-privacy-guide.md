## 📌 가이드 개요 (Overview)
개인정보 처리자는 원칙적으로 정보주체의 동의를 받아야 하지만, 법률에 특별한 규정이 있거나 법령상 의무 준수를 위해 불가피한 경우에는 동의 없이도 정보를 제공하거나 수집·이용할 수 있습니다. 본 가이드는 그 구체적인 판단 기준과 관련 법률 예시, 주요 판례를 정리합니다.

## ⚖️ 법적 근거 (Legal Grounds)
1.  **제18조 제2항 제2호**: 다른 법률에 특별한 규정이 있는 경우 동의 없이 **제3자 제공** 가능.
2.  **제15조 제1항 제2호 / 제17조 제1항 제2호**: 법령상 의무 준수를 위해 불가피한 경우 **수집·이용 및 제공** 가능.

## 🔍 '특별한 규정' 판단 기준
해당 법률이 다음 요건 중 하나 이상을 충족해야 합니다:
*   **명확성**: 제공 근거가 명시되어 있고, `"따라야 한다"`, `"제공할 수 있다"`는 표현이 포함된 경우.
*   **강제성**: 자료 제출 거부 시 과태료 등 **제재 조항**이 존재하는 경우.
*   **구체성**: 요청 기관, 제공 정보, 제공 가능성이 법률 취지상 명확히 드러나는 경우.

## 📚 관련 법률 예시 (Reference Laws)
| 법률명 | 주요 내용 |
| :--- | :--- |
| **자동차손해배상보장법** | 국토부장관 요청 시 보유기관의 제공 의무 (제14조) |
| **위치정보법** | 긴급 구조 시 위치정보 제공 요청 불응 금지 (제29조) |
| **의료법 / 병역법** | 진료기록 전송 및 제출 요청에 대한 대응 의무 |
| **전자상거래법** | 거래기록 보존 및 열람 제공 의무 (제6조) |
| **국세기본법** | 세무 기록 보존 의무 (제85조의3) |

## ⚖️ 주요 판례 및 심의 사례 (Precedents)
| 사례 | 판단 결과 |
| :--- | :--- |
| **주택조합 명부 제공** | 주택법상 조합원의 열람권은 정당한 권리로, 정보공개 거부 불가 |
| **체납 자료 요청** | 국세징수법 등에 의한 요청은 '법률에 의한 제공'으로 동의 불요 |
| **공익신고용 영상 수집** | 공익신고자 보호법은 개인정보 보호법에 우선하는 특별한 규정으로 인정 |

## 🚩 실무 적용 가이드
1.  **부당한 침해 여부 검토**: 법률에 근거가 있더라도 정보주체의 이익을 부당하게 침해할 우려가 있다면 제공을 거부할 수 있습니다(제18조 제2항 본문).
2.  **불가피성 입증**: 법령상 의무 이행을 위해 수집할 경우, 해당 정보 없이는 의무 이행이 **현저히 곤란함**을 입증할 수 있어야 합니다.
3.  **최소 수집 원칙**: 법령에서 특정하지 않은 과도한 정보 수집은 지양해야 합니다.
